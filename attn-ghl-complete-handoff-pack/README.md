# ATTN Funnel Developer Handoff Pack

This folder is a complete handoff pack for implementing the approved ATTN landing page inside Go High Level.

## Included
- `/code/index.html` — full landing page HTML structure.
- `/code/styles.css` — complete styling file.
- `/code/ghl-custom-css.css` — same CSS, named for direct GHL use.
- `/assets/...` — all key design assets, placeholders, backgrounds, reference screens, and reusable small elements.
- `/docs/...` — implementation, brand, and asset guidance.

## Quick start
1. Upload the assets in `/assets` to your Go High Level Media Library.
2. Build the page in a custom HTML section or recreate the sections natively in GHL.
3. Paste the styles from `ghl-custom-css.css` into Page Settings > Custom CSS.
4. Replace demo forms with native GHL forms.
5. Keep the hero section as a **video block** and use `hero-video-cover.png` as the poster image.
