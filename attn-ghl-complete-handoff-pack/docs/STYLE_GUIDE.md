# Style & Design System Guide

## Overall visual direction
Vintage-meets-futuristic.
- Professional first.
- Retro-futuristic second.
- Electric details should support the design, not overwhelm it.

## Core colours
- Primary orange: `#FF5C1B`
- Dark orange: `#D24713`
- Electric blue: `#1E92FF`
- Dark base: `#05070A`
- Dark surface: `#0D1218`
- Paper / warm white: `#F5F1E8`
- Cream accent: `#EFE7DA`

## Fonts
These are web-safe recommendations that match the approved direction:
- **Display / headings:** Teko
- **Body / UI:** Inter
- **Handwritten notes / arrows:** Caveat

### Optional premium alternatives
If the team wants a closer premium design feel, test:
- Display: Eurostile Extended / Neue Machina / Bebas Neue
- Body: Neue Montreal / Inter
- Handwritten annotations: Caveat / Kalam

## Small recurring design elements
### Arrows
Use the supplied SVG arrows for handwritten directional notes.
- `arrow-orange.svg`
- `arrow-blue.svg`

### Sticky tape
Use the supplied tape SVG assets for founder photo blocks, image notes, or “polaroid” style content moments.
- `sticky-tape-horizontal.svg`
- `sticky-tape-corner-left.svg`
- `sticky-tape-corner-right.svg`

### Backgrounds
- Dark sections use `bg-dark-tech.png`
- Light sections use `bg-light-blueprint.png`

### Visual balancing rule
Use alternating black / white sections, with orange form blocks to punctuate the page and improve CTA visibility.
